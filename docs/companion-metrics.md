# Companion metrics (not product)

Product metric: **TRUST-INTEGRITY** (catch / residual / false_block / verify_ok).

Companion lanes (livability / cost only — never pooled into product headlines):

- Utility task success (Harbor TB2 / Claw mean reward)
- deny_by_reason histogram of shell reason codes
- token_ratio ON versus OFF (gated cost)
- Timeout count (AgentTimeoutError ON versus OFF; non-increasing after scope fixes)
- Trust-shaped SWE subset or claim-grader companion (future; not task-success alone)

Any published number must live in `docs/claims.json` with a MEASURED or EXTERNAL tag.
