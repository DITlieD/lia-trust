# LIA Trust Kernel v0.2.2

## Fixes

- **Install default scope**: empty `--allowed-root` now defaults to `$HOME` instead of the
  install-time cwd. Installing from a single repo clone no longer shrinks the gate to that
  repo and false-denies ordinary agent shell/read paths under the user home (Grok/Claude).
- **Reinstall preserves roots**: when no explicit `--allowed-root` is given, an existing
  `~/.lia-trust/config.json` `allowed_roots` list is kept so reinstall does not narrow a
  broader home-scoped install.

## Includes

- All of v0.2.1 (Grok camelCase / toolName / target_file envelope support in the Claude adapter).
